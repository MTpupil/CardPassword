## 模块分解

### 登录功能

登录的实现是接收用户的账号和密码，通过与数据库中的数据进行比对，判断账号是否存在以及是否正确。

登录失败会判断失败原因：

1. 用户权限不足（仅admin用户才可以登录后台）
2. 账号或密码错误

失败后自动返回登录界面，并提示用户失败原因。
